# 🚀 Apoorva's Developer Journey
A cinematic, scroll-driven timeline built for the E-Cell BMSITM Tech Team Recruitment (2026-27).

**🌐 Live Demo:** [View on Vercel](https://my-journey-git-main-apoorva-jet04s-projects.vercel.app/)

## 🌟 The Concept
This isn't just a portfolio; it’s an honest story. It tracks my transition from a pre-medical aspirant (NEET) to a 2nd-semester Information Science student at BMSIT&M.

## 🛠️ The Tech Stack
**Frontend**: Next.js 14, Tailwind CSS

**Animations**: Framer Motion (Scroll-driven canvas crossfades)

**Design Strategy**: Vibe Coding — leveraging AI for high-end asset generation and rapid UI prototyping.

## 🎨 UI/UX Highlight
The background features a 'Neural Core' that morphs from an organic biological state into a technical AI-driven node as you scroll, symbolizing my growth and my ultimate goal: Agentic Generative AI.
